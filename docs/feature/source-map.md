# 端游资料源记录

用途：记录已用于理解 `feature` 的端游资料源，后续做系统复刻时优先从这些源继续核对细节。

## 本地美术参考图

入口：

- `docs/reference/art/index.md`：按分类查看地图、装备、技能标识、角色形象、宠物形象等本地参考图。
- `docs/reference/art/catalog.json`：机器可读目录，保留来源 URL、抓取日期、可信度、本地路径和版权说明。
- `docs/reference/art/image-index.csv`：表格化图片索引。
- `docs/reference/art/gaps.md`：失效图片、来源缺口、需人工复核项。
- `docs/reference/assets/localized/`：本地化图片文件，按 `map_scene`、`equipment_item`、`skill_icon`、`role_character`、`pet_character`、`pet_system`、`dungeon_activity` 等分类存放。

使用边界：

- 图片仅作内部研究、风格分析和功能对标参考。
- 版权归腾讯/原站或相应权利人，禁止直接复用为游戏资产。
- 后续产品美术必须原创重绘；可读取本地图片做轮廓、题材、系统表现和图标语义参考。
