# 功能 feature 知识底座

目标：为“QQ自由幻想端游精神复刻，本地 iPhone 单人自玩版”保存功能范围、资料来源和制作拆解。

当前原则：

- 只做本地单人自玩，不做社交、玩家交易、多人竞争。
- 对标端游玩法结构和体验，不复制名称商标、美术资源、音乐、文本、客户端资源或服务端协议。
- 后续所有方案、排期、原型都以 `feature.md` 为范围基准。

文档：

- `feature.md`：Chris 已确认的保留功能清单。
- `source-map.md`：端游资料源和已阅读范围。
- `production-brief.md`：个人工作室视角的系统拆解，覆盖产品、策划、数值、美术和开发注意点。
