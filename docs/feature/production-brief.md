# Feature 制作拆解

视角：个人工作室，目标是本地 iPhone 单人自玩版。

核心判断：不要追“全量 MMO 复刻”。应精准复刻三件事：职业刷怪手感、装备/宠物养成深度、平行世界和日常带来的长期目标感。

美术参考：端游公开网页中的地图、装备/道具、技能标识、角色职业、宠物形象等参考图已本地化到 `docs/reference/art/index.md` 和 `docs/reference/assets/localized/`。这些图片只用于内部研究、风格分析和功能对标，版权归腾讯/原站或相应权利人，禁止直接复用；本项目实际游戏资产必须原创重绘。

## 1. 基础体验

端游细节理解：

- 角色从创建开始选择性别、职业、发型/发色，职业包括战士、剑客、刺客、术士、药师。
- 10级后通过职业任务转职；后期存在飞升，达到等级和任务条件后进入更高职业阶段。
- 端游是鼠标点地移动、F键技能栏、自动寻路、自动打怪、自动补给、任务指引组合。

单机手游改造：

- iPhone端采用虚拟摇杆或点地移动二选一，首版建议虚拟摇杆+点击目标。
- 技能栏改为普攻、4个主动技能、1个药水/补给按钮、1个自动战斗开关。
- 自动寻路只在主城和任务地图内工作，跨地图用传送点。
- 自动战斗要支持优先怪物、血蓝阈值补给、背包满停止、死亡自动回城。

制作要点：

- 产品：开局30秒进入战斗，5分钟完成升级和换装，15分钟遇到第一个精英怪。
- 策划：五职业都保留，但首版先做1个近战、1个远程、1个治疗模板，再扩展到五职业。
- 数值：前30级升级快，之后用装备、宠物、副本门槛制造目标。
- 美术：原创Q版2D/2.5D角色，职业轮廓必须清晰，不能使用端游原资产。

## 2. 战斗刷怪

端游细节理解：

- 核心循环是野外刷怪、任务推进、精英/BOSS挑战、平行世界副本。
- 职业定位明确：战士偏物理输出，剑客偏均衡/生存，刺客偏敏捷爆发，术士偏法术群攻，药师偏治疗/辅助。
- 普攻、主动技能、被动技能共同决定战斗节奏。

单机手游改造：

- 所有副本默认单人可过，必要时用宠物/幻神承担队友功能。
- 药师不能只做辅助，必须给单机输出路线。
- BOSS战保留走位、技能预警、阶段召唤、破防窗口。
- 死亡复活做轻惩罚：回城/扣耐久/损失少量金币三选一，避免单机挫败。

制作要点：

- 产品：挂机爽感要强，但BOSS要让玩家手动更划算。
- 策划：怪物分普通、精英、首领、活动怪；掉落分别对应经验、材料、装备、稀有卡片。
- 数值：战斗公式先简单，保留攻击、防御、命中、闪避、暴击、属性伤害、治疗强度。
- 美术：怪物按地图主题做家族化设计，精英怪用变色和特效区分。

## 3. 地图任务

端游细节理解：

- 主城承担NPC、商店、合成炉、任务、传送、宠物商人等功能。
- 新手村和野外地图负责等级推进，任务包括主线、支线、职业、收集、循环和平行世界任务。
- 怪物图鉴、NPC和任务列表构成探索和目标引导。

单机手游改造：

- 主城菜单化与场景化结合：保留可走主城，但复杂功能可以通过底部菜单直达。
- 地图数量首版压缩：1主城、1新手村、3野外、2副本足够形成闭环。
- 任务指引要强，避免端游式大量跑腿在手机上变疲劳。

制作要点：

- 产品：任务不是文本阅读核心，而是解锁系统和地图的节奏器。
- 策划：每张野外图绑定1个主线节点、2种材料、1个精英怪、1个图鉴目标。
- 数值：地图等级跨度控制在5-8级，避免过快淘汰。
- 美术：地图主题可原创化，如桃源村、龙城风格主城、竹林、矿洞、雪域、遗迹。

## 4. 装备道具

端游细节理解：

- 装备包括职业外套、武器、防具、饰品、套装、外甲。
- 道具包括收集品、消耗品、卡片；卡片可镶嵌到有孔装备中。
- 装备系统包含耐久/修理、加工、打孔镶嵌、升级、拆解、附魂和基础属性。
- 端游合成/加工存在成功率、材料、提升概率道具等机制。

单机手游改造：

- 保留随机掉装、品质、孔位、卡片、强化、附魂，耐久弱化为BOSS/死亡后才损耗。
- 装备加工失败不建议直接爆装，单机版更适合失败消耗材料或掉强化等级。
- 拆解必须成为材料回收主通道，支撑长期刷怪。

制作要点：

- 产品：装备成长是核心爽点，掉落光柱、比较面板、一键换装必须有。
- 策划：装备部位建议武器、衣服、帽子、手套、鞋子、项链、戒指、外甲。
- 数值：品质白/绿/蓝/紫/橙，孔位0-4，强化+1到+15，附魂提供流派词条。
- 美术：首版可用原创图标+纸娃娃简化，不必一开始做全套换装展示。

## 5. 宠物系统

端游细节理解：

- 宠物可捕捉、喂养、召唤，有饥渴度、信赖度、经验、成长类型、技能。
- 宠物成长类型影响力量、智力、敏捷、体力等倾向。
- 信赖度受喂养、召唤、死亡、交易等影响；单机版去掉交易影响。
- 宠物技能包含生活技能、战斗辅助和抗性类技能；技能丸提高领悟概率。
- 宠物可幻化、培育、装备、进入宠物法阵。

单机手游改造：

- 宠物成为单机队友：可输出、承伤、回血、拾取、骑乘或提供被动。
- 喂养不要太频繁，改为每日/副本前补给。
- 培育系统保留一代到二代，但双人培育改为单人双宠培育。
- 宠物法阵作为后期全局属性系统，不放首版。

制作要点：

- 产品：宠物捕捉是情绪点，首版必须做“怪物低血量捕捉”。
- 策划：宠物分草食/肉食/杂食可保留为喂养差异，但不要做太重。
- 数值：宠物等级不能超过玩家太多，建议超过玩家5级后属性衰减或不可出战。
- 美术：宠物比角色更能承载可爱感，优先投入小体型、清晰轮廓、动作少但反馈强。

## 6. 副本活动

端游细节理解：

- 平行世界是端游副本主线，包含赤枭巢穴、八仙结界、纯阳宝塔、塔狱、天空之泉、天之眼、失落神殿等。
- 后期副本包括花妖巢穴、夜影村、迷雾沼泽、冰火双螺、灵泽殿、先祖山、八仙东海战、妖皇府、昆仑雪、通天魔劫塔。
- 副本往往绑定等级门槛、任务领取、信物/道具进入、BOSS和特定产出。

单机手游改造：

- 所有副本改成“单人挑战+宠物/幻神助战”。
- 信物进入可以保留，但来源改为野外掉落或任务奖励。
- 副本入口集中在主城“平行世界使者”，减少手机端跑图。

制作要点：

- 产品：副本承担装备和稀有宠物材料产出，不能只是剧情关卡。
- 策划：每个副本有一个主机制，一个目标产物，一个首通奖励，一个重复刷奖励。
- 数值：副本难度分普通/困难/噩梦，噩梦产出高阶附魂、卡片、宠物材料。
- 美术：副本主题要强，首版只需2-3套地图组件复用。

## 7. 单机日常

端游细节理解：

- 试炼、珍珠海域、保卫战、天魔劫、通天塔、怪物猎杀令构成日常活动池。
- 这些活动本质是限次、限时或目标明确的奖励玩法，用来分配经验、材料、稀有产出。

单机手游改造：

- 不做真实日期强制绑定，改为每日重置或本地冷却。
- 怪物猎杀令适合做成刷怪任务板。
- 通天塔适合做爬塔和战力检验。
- 珍珠海域/保卫战/天魔劫可分别承载金币、材料、装备、经验产出。

制作要点：

- 产品：日常不能太多，首版保留3个入口即可。
- 策划：日常产出要互补，避免玩家只刷一个最优活动。
- 数值：奖励设计要支撑“10分钟短玩”和“1小时刷爽”两种节奏。
- 美术：活动入口统一用图标和列表，不必每个活动单独做复杂UI。

## 8. 后期养成

端游细节理解：

- 后期养成包括幻神、法宝、灵、战魂、典籍卡、称号、110级新等级、装备祝福、龙翼秘宝、翅膀雕花。
- 这些系统多数是长期属性成长、套装/收集加成、外观强化或高级材料消耗。

单机手游改造：

- 首版只预留数据结构，不急着开放。
- 幻神可作为第二伙伴系统，与宠物区分：宠物偏常驻助战，幻神偏爆发/变身/大招。
- 法宝/灵/战魂/典籍卡都可以抽象成“插槽+升级+套装效果”。
- 称号做成成就奖励，兼顾收集和属性。

制作要点：

- 产品：后期系统要逐步解锁，不能开局全堆给玩家。
- 策划：每个后期系统必须有独立产出来源和清晰养成目标。
- 数值：避免无限乘区膨胀，首版统一为基础属性、战斗属性、特殊效果三类。
- 美术：龙翼和翅膀雕花视觉成本高，放后置；称号和典籍卡成本低，可先做。

## 9. 首版切分建议

P0必须可玩：

- 角色创建
- 1-3个职业模板
- 等级/属性/技能
- 1主城、1新手村、3野外
- 普攻、主动技能、自动战斗、死亡复活
- 基础装备、掉落、背包、穿戴、消耗品
- 任务指引、主线/收集/循环任务
- 1个宠物捕捉与出战
- 2个单人副本

P1补核心味道：

- 五职业完整
- 转职
- 装备加工、打孔镶嵌、卡片、拆解、附魂
- 宠物喂养、信赖、技能丸、培育
- 怪物图鉴
- 试炼、通天塔、怪物猎杀令

P2补长期目标：

- 飞升
- 全平行世界副本框架
- 套装、外甲、装备祝福
- 宠物装备、宠物法阵
- 幻神、法宝、灵、战魂、典籍卡、称号

P3补内容量：

- 全部副本活动产出
- 110级新等级
- 龙翼秘宝
- 翅膀雕花
- 更多地图、怪物、BOSS、装备外观
