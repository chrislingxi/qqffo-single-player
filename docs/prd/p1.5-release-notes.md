# P1.5 落地说明

更新时间：2026-07-04

## 上线范围

P1.5 已完成本地 H5 可玩版，入口仍为项目根目录 `index.html`，本地开发服务为 `http://localhost:5173`，同 Wi-Fi iPhone 可访问 `http://192.168.1.2:5173`。

## 对标改动

- 职业：五职业保留，职业定位、武器、每级自动配点接入腾讯官方旧资料。
- 技能：P1 子集替换为官方技能名、等级和可确认效果，包括战士连环枪法/穿刺之枪/破防之枪、剑客罡气剑/真空剑/雷霆之怒、刺客倾力一击/狙击/淬毒术、术士火弹术/冰箭术/火球术、药师光弹术/治愈术/祝福术。
- 美术：原版怪物参考图已落地到 `assets/reference/original/`，运行时使用 `assets/game/qstyle/` 的 Q 版资产。
- 地图：从抽象网格改为有地形色块、路径、装饰和怪物分布的竖屏场景。
- 交互：首屏改为游戏 HUD，含角色状态、小地图、任务追踪、技能轮盘、自动寻路、自动战斗、自动补给。
- 验证：数据烟测、五职业挂机模拟、真实浏览器 iPhone 视口烟测均通过。

## 不能冒充原版的字段

- MP、冷却、施法距离、技能图标、部分状态效果。
- 属性换算公式、等级经验曲线。
- 怪物血量、攻击、防御、经验。
- 1-35 级装备属性与图标。
- 官方地图原图、职业男女形象、部分 P1 怪物原图。

这些字段已记录到 `docs/prd/p1.5-source-gaps.md`，后续继续按“先补资料，再结构化，再开发”的顺序推进。
