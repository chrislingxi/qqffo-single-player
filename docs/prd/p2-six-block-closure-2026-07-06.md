# P2 六块体验闭环记录

日期：2026-07-06

## 对齐目标

本轮只围绕用户重新定义的 6 块推进：

1. 地图个性化。
2. 怪物生态刷新。
3. 技能特效模板。
4. 图形 icon。
5. 宠物强系统面板。
6. 任务传送与跨地图寻路闭环。

## 已落地

- 地图：13 张地图场景配置，覆盖石板/草地/泥路/水域/法阵地面、城墙/竹桥/荷塘/树丛/建筑/传送氛围、雨/灵光/火星/草叶等氛围；新增可走区域修正，水边和副本墙体不再只是装饰。
- 怪物：刷新区从 11 个扩到 19 个，关键野外地图都有多个 `spawnZone`；每个 zone 有怪物种类、数量、刷新时间、活动半径；精英区用 `maxAlive: 1` 和更长刷新时间表达。
- 技能：新增/强化火、冰、毒、光、斩击、治疗、宠物爪击、范围法阵模板；buff/debuff 有小状态 icon。
- 战斗可读性：伤害飘字按敌人实际坐标右侧生成，并加入同区域错层避让，降低怪名、伤害、技能名挤在一起的问题。
- Icon：顶部系统入口和快捷按钮改为 SVG 图形 icon，任务卷轴、背包袋、地图卷、宠物爪、日常令牌、宝塔、武器、药瓶不再使用 `卷/囊/图/宠/令/塔` 文字作为 icon 本体。
- 宠物：宠物面板升级为“左侧模型 + 生命/法力/等级/饥渴/信赖 + 攻防命中魔攻魔防 + 六维加点 + 战斗/生活/法阵/装备技能格 + 我的宠物列表”。
- 任务：跨图任务仍通过 `questWarp` 先传送到目标地图，再自动寻路到 NPC/怪物/副本；目标点会经过可走区域修正。

## 模拟自由幻想玩家体验反馈

本轮后，“按钮还是文字”“宠物只是卡片”“怪只在身边临时刷”“地图仍是同一张底图”的问题已经明显改善。当前更像一个横屏 RPG 手游框架，能看到地图氛围、刷新区、宠物出战、技能元素和正式系统入口。

仍需后续继续做：

- 战斗中心已做飘字错层和敌人坐标跟随，但高暴击大字和 BOSS 名仍会短时间同屏，后续可继续做完整飘字队列和战斗文字层级。
- 地图仍是程序化 Q 化场景，不是最终 AI 高质量逐张地图成图。
- P2 审计仍提示五职业技能树深度不足，以及装备/宠物/怪物数值来源不足。
