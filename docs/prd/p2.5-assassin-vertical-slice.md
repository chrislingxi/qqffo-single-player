# P2.5 刺客 1-20 级极致纵切 PRD

## 定位

从 P3 全量原型切换为 `2.5D Q版商业手游 Demo` 纵切：先只打磨刺客 1-20 级体验，让玩家在 iPhone 横屏/PWA 中完成创建角色、接任务、传送、寻路、刷怪、升级、换装、宠物、20级前小Boss试炼的闭环。

## 本版范围

- 职业：仅开放刺客；其他职业保留但创建页锁定。
- 等级：纵切体验封顶 20 级，避免继续滑入 P3 大而全。
- 地图：桃源村、芳草林、阳原平原中、花妖巢穴。
- 战斗：倾力一击、狙击、淬毒术、双刃修炼形成 1/4/8/12 级技能曲线。
- 怪物：粉粉兔、食人花、野猪、野猪头目、花妖头目。
- UI：横屏 HUD、任务追踪、技能轮、背包、宠物、角色/技能、地图面板必须完整可用。

## 体验标准

1. 第一眼必须像横屏手游 Demo，而不是文字/线框网页。
2. 10 秒内玩家能看到角色、任务、地图、怪物或下一步目标。
3. 打怪必须有接近、站位、技能动作、受击、飘字、暴击、死亡、掉落。
4. 怪物在地图上预先可见，按 spawnZone 分布和刷新。
5. 任务跨地图时必须先传送，再自动寻路到 NPC/怪物/副本。
6. 面板打开时游戏不因状态混乱卡住，且不遮挡主要战斗区域到不可玩。

## 后续扩展原则

本纵切达标前，不继续横向扩 P3 全功能。达标后再按同一标准复制到其余职业、地图、副本和后期系统。
