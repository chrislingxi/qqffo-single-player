# P1.5 对标体验重制计划

更新时间：2026-07-04

## 目标

把当前 P1 从“HTML 功能样机”升级为“自由幻想感受样机”。P1.5 不扩 P2/P3 系统，只重做 P1 的数据真实性、美术感受、战斗手感、手游交互和地图体验。

## Chris 已确认原则

1. 完全本地化：所有产品 feature、数值、美术参考先从本地资料库读取；本地没有的，必须先去官方渠道和可信资料源补齐，再进入开发。
2. 美术在原版基础上 Q 化，不完全新造，也不直接复用原版素材进入 runtime。
3. 交互改成游戏 HUD，不再是网页工具面板。
4. 战斗手感重做，优先参考本地原版资料。
5. 地图找本地真实地图资料，没有则继续找原版。
6. 除美术表达层外，产品 feature 和数值目标是完全复刻；查不到的字段标缺口，不用推断值冒充正式数据。

## P1.5 新执行准则

P1.5 的开发顺序必须改为：

1. 原版资料本地化。
2. 缺失资料补采。
3. 字段级审计。
4. 美术 Q 化。
5. 游戏实现。
6. 移动端体验测试。

任何对象进入游戏 runtime 前，都必须具备以下字段：

- `original_name`：原版名称。
- `source_url` 或 `source_file`：资料来源。
- `source_confidence`：来源置信度。
- `value_confidence`：数值置信度。
- `original_asset`：原版美术参考，没有则写 `missing`。
- `q_asset`：Q 化后游戏资产，没有则写 `missing`。
- `q_asset_source`：Q 化资产基于哪个原版素材或资料描述生成。

推断字段只能用于临时技术跑通，不能作为 P1.5 验收通过依据。

## 当前问题

当前 P1 的主要问题不是功能缺，而是“对标度不够”：

- 技能名/等级/部分倍率已接入官方文本；技能消耗/冷却、属性、怪物、装备数值仍大量为推断值。
- 地图是抽象网格，不是原版地图感。
- UI 是网页面板，不是 QQ 自由幻想的游戏界面。
- 角色、怪物、NPC 都是符号化圆点，没有 Q 版纸娃娃感。
- 战斗缺少前摇、后摇、技能特效、飘字、掉落表现。

## P1.5 产出范围

### 数据

- 替换 `data/design/classes.json`
  - 使用官方职业定位和成长方向。
  - 保留临时数值，但字段标记为推断。
- 替换 `data/design/maps.json`
  - 使用原版地图名：桃源村、芳草林、芳草林东、芳草林西、阳原平原、龙城、龙城南郊/北郊等。
- 替换 `data/design/monsters.json`
  - 使用原版怪物名和掉落来源：粉粉兔、小蜗牛怪、食人花、野猪等。
- 替换 `data/design/quests.json`
  - 使用官方新手任务链描述：小试身手、兔毛、新衣、武器师匠的理想、前往龙城等。
- 保留 `source_url`、`source_file`、`source_confidence`、`stat_confidence`。

### 美术

- 所有美术素材先进入 `assets/reference/original/`，记录来源。
- 所有游戏内美术素材进入 `assets/game/qstyle/`，必须关联原版参考。
- 角色采用五职业原版形象参考后的 Q 版纸娃娃。
- 怪物采用原版怪物/宠物图或官方描述后的 Q 化版本。
- 地图采用原版地图/小地图参考后的 Q 化 tilemap，不使用抽象网格代替。
- UI 采用原版界面语言的手游化重绘：木质、卷轴、金边、快捷栏、小地图、任务追踪。

### 交互

- 手机端主界面改为：
  - 左上角色血蓝。
  - 右上小地图。
  - 左侧任务追踪。
  - 左下摇杆/点击移动。
  - 右下普攻和技能按钮。
  - 底部背包、角色、技能、宠物、地图入口。
- 背包/角色/宠物等改成弹窗，不再占半屏 tab 工具面板。

### 战斗

- 普攻有攻击间隔、前摇、受击飘字。
- 主动技能有图标、冷却转圈、特效、蓝耗。
- 怪物死亡有掉落光柱。
- 自动战斗表现为角色寻找附近怪并移动攻击。

## P1.5 验收标准

- 手机打开后第一眼像游戏，不像网页表单。
- 桃源村、芳草林、龙城等名称和任务来自本地官方资料。
- 至少一条 1-15 级新手路线按原版任务名和怪物名跑通。
- 五职业定位文本来自官方资料。
- 未查到的技能/属性/装备数值全部进入缺口表；技术占位值不能算验收完成。
- 游戏中出现的角色、怪物、宠物、装备、技能、地图、NPC、UI 元素都有原版参考或明确缺口记录。
- 已有原版素材的对象必须展示 Q 化资产，不能继续用圆点/线框。
- iPhone 竖屏可以完整创建角色、接任务、自动寻路、打粉粉兔、掉兔毛、交任务、进入龙城。
