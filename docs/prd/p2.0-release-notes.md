# QQ自由幻想单机 P2.0 交付说明

## 定位

P2.0 是本地 iPhone 横屏 PWA 的个人可用版。目标不是继续做文字线框原型，而是在 P1.8 横屏 HUD 基础上补齐中期可刷循环、AI Q 版运行时美术、PWA 缓存更新和自动化验收。

2026-07-05 复盘修正：当前实现不能继续按“P2.0 已完成”理解。真机体验暴露出战斗布局混乱、按钮和 icon 堆叠、角色/怪物层级错误、自动战斗交互生硬、技能/普攻/暴击反馈不足、职业属性/技能/宠物等数值与原版资料不一致等问题。P2.0 后续必须按 `mobile-gameplay-standards.md` 和 `p1-p3-prd.md` 的新增条款重建体验与数据链路。

## 本版新增

- 美术：新增 `assets/game/qstyle/ai/qqffo-qstyle-concept-sheet-v1.png` 作为商业 Q 版风格标尺，并切出 10 个运行时 PNG。
- 资产规范：运行时怪物必须有 Q 版资产映射；尚未单独产出的怪物用 `ai_q_runtime_alias_v1` 标明，不伪装成官方完整素材。
- 成长：等级上限从 35 提升到 55，新增 35-55 中期地图 `灵泽荒原`。
- 装备：新增加工 +9、打孔镶嵌、附魂，装备属性会实际进入战斗公式。
- 宠物：新增培育、技能丸、宠物法阵，宠物技能等级和法阵会进入战斗属性。
- 转职/飞升：新增职业进阶和飞升节点，消耗金币/材料并提供永久属性。
- 日常：新增试炼、怪物猎杀令、通天塔、珍珠海域的单机日常入口。
- 副本：新增 `灵泽殿·单机`、`龙翼秘宝·单机`，P2 五职业模拟覆盖 4 个副本。
- PWA：缓存升级到 `ffo-pwa-v2`，新增数据表和 AI PNG 资产进入离线核心包。

## 验收结论

旧结论“个人可用、可横屏刷、无明显基础体验问题”已撤回。当前只能视为 P2 方向样机：已有部分中期系统和 Q 版资产方向，但尚未达到 P2.0 个人可用/近商业 UI/30 分钟任务牵引/原版数值对齐的完成标准。
