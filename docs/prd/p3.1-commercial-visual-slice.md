# P3.1 商业视觉纵切目标

日期：2026-07-18

## 目标

P3.1 不扩玩法功能，专门对齐商业手游视觉体验标准：全屏地图资产感、首页/HUD icon 质感、右上小地图精细度、战斗打击动态稳定交付。

## 对标标准

- 地图：每张地图有独立地貌、道路、水域、地标、前景遮挡和氛围光，不再只靠程序化底纹。
- UI/Icon：首页和游戏内系统入口统一金边圆形图标，具备材质、高光、按下态、红点和短 label。
- 小地图：右上角必须像真实 RPG 小地图，包含地形底、道路、水域、怪物点、目标点、玩家箭头。
- 战斗：刺客普攻/技能必须体现接敌、位移、前摇、命中爆点、受击闪白、击退、hit stop、飘字、掉落爆点。
- 稳定：打击动效必须是代码内稳定模板，不依赖偶然截图；发布前由自动门禁和浏览器横屏自测共同验收。

## 本版落地范围

- `DATA_VERSION` 升级到 `p31-visual-01`，避免 iPhone PWA 继续读取旧包。
- 地图渲染新增商业视觉层：AI 横屏底图叠加、径向光照、前景暗角、动态水波、草叶光斑。
- 小地图新增地形切片：water/path/terrain/monster/player/target，不再只是圆形网格。
- 战斗新增打击模板：`slashArc`、`impactSpark`、`shockwave`、`afterimage`、`screenShake`、hit stop 与怪物 knockback。
- UI 追加 P3.1 final visual lock：顶部 icon、技能轮、聊天栏、左侧任务入口统一成商业手游质感。
- 新增 `test:p3-visual`，并接入 `publish:pages`。

## 验收方式

- 自动测试：`verify:p2-six`、`verify:vertical-slice`、`verify:p3-m4`、`test:ui-experience`、`test:p3-visual`、`test:smoke`、`test:vertical-player`、`test:player`。
- 浏览器：844x390 横屏打开，确认 HUD 无重叠、小地图/技能轮/任务追踪可读、战斗中可见打击特效。
- 玩家模拟：自由幻想玩家视角检查“像不像、爽不爽、是否卡顿、是否低级、是否迷路”。
