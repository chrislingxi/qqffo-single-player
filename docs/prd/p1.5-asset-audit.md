## 审计规则

- `official_original`：已有官方原版资料或原图。
- `localized_reference`：已保存到 `assets/reference/original/` 或 `docs/reference/`。
- `q_ready`：已有 Q 化 runtime 资产。
- `missing_source`：本地缺资料，必须先补采。
- `inferred_value`：技术占位值，不能算 P1.5 正式验收。

## 结论

P1.5 当前可以推进“感受重制”，但正式验收前必须补齐：

- 五职业技能图标、消耗、冷却、施法距离和升级曲线。
- 属性/经验/装备/怪物数值。
- 五职业角色原版图。
- 桃源村、芳草林、龙城地图原图。
- P1 已出现怪物的原版图。
